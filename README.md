# Job Portal Application

A Spring Boot based Job Portal web application.

## Features
- User Registration & Login
- Add Job
- View Jobs
- Apply for Jobs
- Admin Management

## Technologies Used
- Java
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL
- Maven

## How to Run
1. Clone the repository
2. Import into STS/Eclipse
3. Run JobportalApplication.java
4. Open http://localhost:8080
